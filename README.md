# Strapi Learning Project

这是一个用于学习 Strapi CMS 的项目。我们将逐步构建一个完整的 Strapi 应用，每个重要的步骤都会有 Git 提交记录。

## 项目目标

- 学习 Strapi CMS 的基本概念和使用方法
- 创建内容类型和管理内容
- 了解 Strapi 的权限系统
- 学习如何自定义 Strapi
- 掌握 Strapi 的 API 使用

## 开发环境要求

- Node.js (建议使用 LTS 版本)
- npm 或 yarn
- 数据库 (SQLite, PostgreSQL, MySQL 等)
